# Support Guide

Welcome to the NGXSMK Wallet support guide. We want to ensure you have the best experience using our privacy-first digital vault. Here is how you can find help, report issues, or get in touch.

## 1. Documentation & Self-Help

Before raising an issue, please check our official resources:
- **Official Website**: [ngxsmk.com](https://ngxsmk.com) for updates, product guides, and news.
- **Project README**: Refer to the [README.md](./README.md) for quick-start guides, architecture overviews, and system requirements.
- **Frequently Asked Questions (FAQ)**: Look through existing GitHub issues to see if your question has already been answered.

## 2. GitHub Issues

We use GitHub issues to track bugs and feature requests. 

- **Bug Reports**: If you find a bug in the application, please open a Bug Report. Be sure to provide details about your operating system, version of the app, and steps to reproduce the issue.
- **Feature Requests**: Have an idea to make NGXSMK Wallet better? Submit a Feature Request with a description of what you'd like to see and why.

> [!CAUTION]
> **Do NOT post sensitive information** (like master passwords, private recovery keys, database files, or screenshots containing real credentials) in GitHub issues. All GitHub issues are public.

## 3. Email Support

For account-related queries, licensing, or non-public questions, you can contact us directly via email:
- **General Support**: **[ngxsmk@gmail.com](mailto:ngxsmk@gmail.com)**
- **Security Vulnerabilities**: Please email **[ngxsmk@gmail.com](mailto:ngxsmk@gmail.com)** (see [SECURITY.md](./SECURITY.md) for details).
